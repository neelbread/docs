Get the Bread SDK installed and authenticated so you can start baking AI models.

<Note>
New to baking? Start with [Understanding Baking](/understanding-baking) to learn the core concepts.
</Note>

## Installation

Install the Bread SDK via pip:

<CodeGroup>

```bash MacOS/Linux
pip install aibread
```

```cmd Windows (Command Prompt)
pip install aibread
```

```powershell Windows (PowerShell)
python -m pip install aibread
```

</CodeGroup>

### Optional: Async Performance

For improved async performance, install with aiohttp support:

<CodeGroup>

```bash MacOS/Linux
pip install aibread[aiohttp]
```

```cmd Windows (Command Prompt)
pip install aibread[aiohttp]
```

```powershell Windows (PowerShell)
python -m pip install "aibread[aiohttp]"
```

</CodeGroup>

Then enable it when creating the async client:

```python
from aibread import AsyncBread, DefaultAioHttpClient

async with AsyncBread(
    api_key=os.environ.get("BREAD_API_KEY"),
    http_client=DefaultAioHttpClient()
) as client:
    repos = await client.repo.list()
```

<Tip>
**Requirements:** Python 3.8+ is required. httpx is included automatically.
</Tip>

## Set Up API Key

Set your API key as an environment variable:

<CodeGroup>

```bash MacOS/Linux
export BREAD_API_KEY='your_api_key_here'
```

```cmd Windows (Command Prompt)
setx BREAD_API_KEY "your_api_key_here"
```

```powershell Windows (PowerShell)
$env:BREAD_API_KEY='your_api_key_here'
```

</CodeGroup>

<Tip>
Use [python-dotenv](https://pypi.org/project/python-dotenv/) to manage environment variables in a `.env` file and keep credentials out of source control.
</Tip>

## Initialize the Client

Create a Bread client to interact with the API:

<CodeGroup>

```python Sync Client
import os
from aibread import Bread

client = Bread(api_key=os.environ.get("BREAD_API_KEY"))

# Test connection
repos = client.repo.list()
print(f"Connected! Found {len(repos.items)} repositories")
```

```python Async Client
import asyncio
import os
from aibread import AsyncBread

async def main():
    async with AsyncBread(api_key=os.environ.get("BREAD_API_KEY")) as client:
        repos = await client.repo.list()
        print(f"Connected! Found {len(repos.items)} repositories")

asyncio.run(main())
```

</CodeGroup>

## Verify Your Setup

Test your connection with a health check:

```python
health = client.health.check()
print(f"Status: {health.status}")
print(f"API Version: {health.version}")
```
